#### 大O表示法

常见的大O表示法

* O(log2n) 对数时间 例如二分查找 
* O(n) 线性时间例如简单查找
* O(nlogn) 快排算法
* O(n2) 选择排序
* O(n!)  阶乘时间，图算法中的旅行商问题
